# From Park to Pitch: Applying (Un)Supervised Learning to MLB Analytics
### Alex Yoon, Justin Tseng, Caroline Gaede
### SIADS 696 (Milestone II) - Spring 2026

## Code Description
Supervised learning code can be found in the supervised.ipynb Jupyter Notebook file. Unsupervised learning code can be found in the unsupervised.ipynb Jupyter Notebook file.

## Data Source
All data for the project was downloaded from [Baseball Savant](https://baseballsavant.mlb.com/) via the [pybaseball package](https://github.com/jldbc/pybaseball). NOTE: Data files were larger than 25 MB and are not included in this repository.
